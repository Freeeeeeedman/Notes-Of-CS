1. C++知识

   - 《STL 源码剖析》+侯捷视频
   - 《深入理解 C++对象模型》，看前几章应该就够了

2. 数据结构与算法

3. 操作系统
   - 《深入理解计算机操作系统》
4. linux 
   - 《深入理解 Linux 内核》可以结合操作系统课程对照着来看
   - 《UNIX 环境高级编程》 堪称圣经的 APUE，介绍的是 unix 系统的基本原理和系统调用，可以说是必看
   - 《Linux 多线程服务端编程:使用 muduo C++网络库》介绍了如何使用 C++开发高性能服务器
5. 计算机网络

6. 数据库与缓存

     - 《MySQL 技术内幕 InnoDB 存储引擎》
   - Redis
     - 《redis设计与实现》

7. 设计模式
